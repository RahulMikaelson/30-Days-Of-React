What is a module?

A module is code related to that file. which can be exported and used any-where in project

What is a package?

Package is a collection of module which can be used in another application

What is the difference between module and package

Module is a file that contains code related to that file, whereas package consist of different types of modules basically it a collection of modules.

What is a NPM?

NPM stands for Node Package Manager. It is used to download the necessary or required packages to the application.

How do you create a React Project?

What are the files and folders inside a project folder(package.json, package-lock.json or yarn.lock, .gitignore,node_modules and public)?

package.json: This contains all the necessary information about the project and all the necessary dependencies(packages) information.
package-lock.json: This file contains all the versions of package that are installed and referred in package.json. This helps use to download only the mentioned packages rather than downloading newer or older version which helps un-necessary package issue.
.gitignore: This file consists of list of file or folder names which we don't want to commit them.
node_modules: This folder consists of all the downloaded packages that we have in package.json file.
public: This folder consist of all the files that can be accessed over the browser.
